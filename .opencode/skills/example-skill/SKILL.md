## What I do
这是一个示例技能，用于展示 opencode skills 的使用方法。

这个技能可以帮助：
- 演示 skills 的基本结构
- 展示 frontmatter 的正确格式
- 提供调用示例

## When to use me
当需要了解 opencode skills 工作方式时使用此技能。

## Usage
在 prompt 中可以这样使用：
```
使用 skill 工具加载 example-skill
```

## Example Commands
- `/skill example-skill` - 加载此技能
- 查看 `.opencode/skills/example-skill/SKILL.md` - 查看技能定义
